# JPGtoPNGConverter
Convert images from JPG to PNG by terminal

# How it works
You simply run this code from terminal
```
Fork this repo
Clone this repo
cd jpgtopngconverter
python jpgtopngconverter.py <name_old_folder> <name_new_folder>
```
